# Описание параметров солнечного инвертора (PH1800)

> **Контекст:** Инвертор находится в **холостом ходу**, батареи подключены, солнечные панели **не подключены**.  
> *Данные актуальны для модели PH1800 и аналогичных гибридных инверторов.*

## ⚡ **Charger Message** *(Параметры контроллера заряда)*

### 🔋 **Режимы работы и состояния**


| **Параметр**       | **Значение**  | **Описание**                                                                                                                          |
| ------------------ | ------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| **Work state**     | SelfTest Mode | Текущий режим работы зарядного устройства. **SelfTest Mode** — автоматическое тестирование системы при запуске или в холостом режиме. |
| **Mppt state**     | Stop          | Состояние MPPT-контроллера (Maximum Power Point Tracking). **Stop** — контроллер не активен (нет входа от солнечных панелей).         |
| **Charging state** | Stop          | Состояние процесса заряда батареи. **Stop** — заряд не выполняется (нет источника энергии или батарея заряжена).                      |


### 📊 **Электрические параметры**


| **Параметр**        | **Значение** | **Описание**                                                                                           |
| ------------------- | ------------ | ------------------------------------------------------------------------------------------------------ |
| **PV voltage**      | 0 V          | Напряжение на входе от солнечных панелей. **0 V** — панели не подключены или нет солнечного излучения. |
| **Battery voltage** | 24.2 V       | Напряжение аккумуляторной батареи. Показывает текущий уровень заряда.                                  |
| **Current**         | 0 A          | Ток заряда/разряда батареи. **0 A** — отсутствие тока (холостой ход).                                  |
| **Power**           | 0 W          | Мощность заряда/разряда батареи. **0 W** — энергия не потребляется и не отдается.                      |

| ----------------- | ------------ | ---------------------------------------------------------------------- |

| **External temp** | 0 °C         | Внешняя температура (возможно, датчик не подключен или не калиброван). |

## 🔍 **Пояснения по режимам работы**

### **SelfTest Mode / SelfTest**

- Автоматический режим проверки системы при включении или в холостом режиме.
- Устройство тестирует все компоненты (реле, датчики, соединения) перед началом работы.

### **Stop (Mppt state / Charging state)**

- Контроллер заряда и MPPT не активны, так как:
  - Солнечные панели **не подключены** (PV voltage = 0 V).
  - Нет солнечного излучения (например, ночное время).

### **Disconnect (Реле)**

- **PV Relay / Inverter Relay / Grid Relay / Load Relay** в состоянии **Disconnect** означает:
  - Соответствующий компонент (панели, инвертор, сеть, нагрузка) **физически или логически отключён** от системы.
  - В данном случае это нормально, так как инвертор работает в холостом режиме.

## 📝 **Примечания**

1. **Холостой ход** — инвертор включён, но не выполняет полезную работу (нет генерации от панелей, нет нагрузки).
2. **0 V / 0 A / 0 W** — типичные значения для неактивных компонентов (панели, инвертор, нагрузка).
3. **ACCUM-параметры** сбрасываются только при полном сбросе устройства или замене аккумулятора (зависит от модели).
4. **Температуры** (24–25 °C) — нормальные рабочие значения для радиаторов и трансформатора.
