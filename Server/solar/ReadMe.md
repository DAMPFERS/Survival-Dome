# Описание параметров солнечного инвертора (PH1800)

**Контекст:** Инвертор подключен к солнечным панелям, аккумулятору и внешней сети. Нагрузка не подключена. Устройство находится в рабочем режиме Grid-Tie, но контроллер заряда (MPPT) временно остановлен. Данные актуальны для модели PH1800 и аналогичных гибридных инверторов.

## 📌 Device Info (Информация об устройстве)

| Параметр | Значение | Описание |
| :--- | :--- | :--- |
| The type of machine | PH1800 | Модель гибридного инвертора/зарядно-инверторного устройства. |
| Hardware version(Inverter) | 1.01.01 | Аппаратная версия инверторного модуля. |
| Software version(Inverter) | 1.17.21 | Версия программного обеспечения (прошивки) инвертора. |
| Hardware version(Charger) | 1.01.01 | Аппаратная версия контроллера заряда (MPPT). |
| Software version(Charger) | 1.17.21 | Версия программного обеспечения контроллера заряда. |
| Protocol Edition | 1.04.14 | Версия протокола обмена данными между устройством и мониторинговой программой. |

## ⚡ Charger Message (Параметры контроллера заряда)

### 🔋 Режимы работы и состояния
| Параметр | Значение | Описание |
| :--- | :--- | :--- |
| Work state | SelfTest Mode | Текущий режим работы зарядного устройства. Несмотря на название, система активна. |
| Mppt state | Stop | Состояние MPPT-контроллера. **Stop** — трекер точек максимальной мощности не активен, заряд от солнца не идет, несмотря на наличие напряжения на панелях. |
| Charging state | Stop | Состояние процесса заряда батареи. **Stop** — заряд не выполняется. |

### 📊 Электрические параметры
| Параметр | Значение | Описание |
| :--- | :--- | :--- |
| PV voltage | 74 V | Напряжение на входе от солнечных панелей. Панели подключены и генерируют напряжение. |
| Battery voltage | 26.9 V | Напряжение аккумуляторной батареи. Батарея заряжена выше номинала (24V система). |
| Current | 0 A | Ток заряда/разряда через контроллер. 0 A — ток не поступает от MPPT. |
| Power | 0 W | Мощность заряда от контроллера. 0 W — энергия от солнца не передается. |

### 🌡️ Температурные параметры
| Параметр | Значение | Описание |
| :--- | :--- | :--- |
| Radiator temp | 26 °C | Температура радиатора зарядного устройства. |
| External temp | 0 °C | Внешняя температура (датчик может быть не подключен или показывать 0 при отсутствии сигнала). |

### 🔌 Состояния реле
| Параметр | Значение | Описание |
| :--- | :--- | :--- |
| Battery Relay | Connect | Реле батареи замкнуто — батарея подключена к системе. |
| PV Relay | Disconnect | Реле солнечных панелей разомкнуто. **Важно:** Несмотря на наличие 74V на входе, физическое подключение панелей к цепи заряда разорвано программно или аппаратно. |

## 🔍 Пояснения по текущему состоянию

### Почему PV Voltage = 74V, но Mppt State = Stop?
На скриншоте видно, что солнечные панели подключены и выдают 74 В, однако:
1.  **PV Relay** находится в состоянии **Disconnect**. Это означает, что цепь между панелями и контроллером заряда разомкнута.
2.  **Mppt state** и **Charging state** равны **Stop**.
**Вывод:** Инвертор "видит" панели, но намеренно или вследствие настроек/условий не использует их энергию в данный момент. Возможно, батарея полна, или включен режим приоритета сети, или это временная пауза MPPT.

### Откуда берутся отрицательные мощности (PInverter -123W, Batt power -93W)?
В данной системе отрицательная мощность означает **потребление**:
*   Инвертор работает в режиме **Grid-Tie** без полезной нагрузки.
*   Он потребляет ~123 Вт на свои внутренние нужды (охлаждение, управление, преобразование).
*   Эта энергия берется частично из сети (~109 Вт) и частично из батареи (~93 Вт), так как батарея разряжается (-3 А).

### Статус нагрузки
Несмотря на то, что **Load relay state = Connect**, параметр **Load current = 0 A** и **Sload = 0 VA** подтверждают, что к выходным клеммам нагрузки ничего не подключено. Реле просто находится в замкнутом положении, ожидая появления потребителя.
